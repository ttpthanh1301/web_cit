# CIT Album Images

Nguồn: album công khai của fanpage `https://www.facebook.com/clbcongnghe.cit/photos_albums`.

## Album folders

- `aeternum/`: 14 ảnh cho AETERNUM và HackerRank nội bộ.
- `vinh-danh-cit/`: 8 ảnh cho Vinh danh CIT.
- `httt/`: 12 ảnh cho HTTT.
- `birthday-with-cit/`: 12 ảnh cho Birthday With CIT.
- `tan-cu-nhan/`: 5 ảnh cho Chúc mừng các tân cử nhân.

Manifest cho code đọc:

- `data/cit-facebook-album-gallery.json`

## Lưu ý

- Các file này đã được lưu local để website không phụ thuộc trực tiếp vào link Facebook CDN.
- Dùng ảnh trong các thư mục con album ở trên cho gallery. Không cần dùng các file `.jpg` nằm trực tiếp trong thư mục `albums/`.
- Nếu cần thêm ảnh cho Camping, bóng giao hữu, tuyển thành viên, hướng nghiệp, Lowcode hoặc IAI Hackathon, cần mở fanpage bằng tài khoản đã đăng nhập hoặc cung cấp link bài/album cụ thể.
