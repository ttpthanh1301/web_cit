## Lưu ý dùng cho website

- Dùng file local trong thư mục này, không hotlink trực tiếp từ Facebook CDN.
- Khi đưa lên hosting nhẹ, nên giữ ảnh dưới khoảng 200-300KB mỗi ảnh nếu có thể.
- Gắn `alt` tiếng Việt cho ảnh, ví dụ: "Hoạt động chúc mừng tân cử nhân của CIT".
